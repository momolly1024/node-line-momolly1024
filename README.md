### 使用說明

##### 可以使用的指令：
#### 輸入
【now_playing】：顯示電影原文名稱

【熱映中】：顯示簡體中文翻譯的電影名稱

【機器人在哪】：顯示設定的開頭及地址，並帶超連結至googlemap

【機器人長相】：傳送一句話及機器人照片(使用imgur網站取得網址)

【簡介】：顯示前五部電影簡介

【推薦】：隨機推薦一部熱映中的電影
